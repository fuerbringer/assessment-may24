# assessment-may24

## Build

Tested on Ubuntu 22.04 LTS. Requirements: build-essential, cmake

```
mkdir build
cd build
cmake ..
```

Executables are in `build/part1` and `build/part2`.

## Run

### Part 1

Unit tests

```
./build/part1/testpart1
```

Demo app

```
./build/part1/part1 part1/resource/98-0.txt 
```
